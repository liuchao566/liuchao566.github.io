java之hashmap原理

###
哈希冲突：开放地址法，散列函数法，链地址法。

###
hashmap由数组和链表组成，采用链地址法解决hash冲突。
数组记录hash值的映射关系，链表记录hash冲突的值。
插入的时候，计算的hash值已有，就会在链头插入该值。
一般没有hash冲突的hashmap，查找数组当前链头的next为空，所有查询很快。
但是碰到有hash冲突的hashmap，需要遍历当前链表，性能下降。

###
hashmap的默认出事容量是16，扩充因子是0.75，
为什么容量都是2的n次幂，因为2的n次幂的二进制全是1，再进行与运算时，运算结果只跟高位有关。
